Photo Watermark Tools — 便携版使用说明
=====================================

【Windows】
  双击 start.bat，浏览器会自动打开工具页面。
  关闭黑色命令行窗口即可停止。
  注意：不要直接双击 index.html，否则无法正常运行。
  若页面标题不是「Photo Watermark & Border」，说明端口被占用，
  请关闭其他开发服务器后重新运行 start.bat。

【macOS / Linux】
  在终端执行: bash start.sh
  （需要系统自带 Python 3）

【说明】
  - 无需安装 Node.js、pnpm 或任何开发环境
  - 整个文件夹可拷贝到 U 盘或其他电脑使用
  - 所有图片处理均在本地浏览器完成，不上传网络
  - 首次使用若浏览器提示防火墙，请允许本地访问

【功能】
  - 照片边框与水印编辑
  - EXIF 自动读取
  - 高分辨率 PNG/JPEG 导出
  - 镜头焦外测试图生成
